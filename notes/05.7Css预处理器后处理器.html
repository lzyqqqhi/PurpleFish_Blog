    # CSS 预处理器与后处理器

    # CSS 预处理器

    # 概念与优点

    CSS 预处理器是扩展了 CSS 功能的语言，使样式表更易于维护和编写，主要优点包括：

    • 变量：定义全局或局部变量，便于统一管理样式属性。
    • 混合（Mixins）：创建可重用的代码块，在不同地方调用，减少代码冗余。
    • 嵌套规则：嵌套 CSS 选择器，使样式结构更清晰，符合逻辑。
    • 条件判断和循环：支持条件语句和循环结构，增强样式动态性。

    # 典型应用场景

    • 大型项目开发：分层拆分、批量管理 CSS 代码，便于团队协作和代码管理。
    • 样式复用：通过变量、Mixins 和函数提高代码复用性。
    • 动态样式：利用条件判断和循环生成动态样式。

    # 代码示例（Sass）

    // 定义变量
$primary-color: #3498db;
$secondary-color: #2ecc71;

// 定义 Mixin
@mixin button-style($color) {
  background-color: $color;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;

  &:hover {
    opacity: 0.9;
  }
}

// 使用 Mixin
.primary-button {
  @include button-style($primary-color);
}

.secondary-button {
  @include button-style($secondary-color);
}

// 嵌套规则
.menu {
  background-color: $primary-color;
  padding: 10px 0;

  .menu-item {
    color: white;
    padding: 5px 15px;
    display: inline-block;

    &:hover {
      background-color: darken($primary-color, 10%);
    }
  }
}

    # CSS 后处理器

    # 概念与优点

    CSS 后处理器在 CSS 编译后对代码进行优化和处理，主要优点包括：

    • 自动添加浏览器前缀：如 Autoprefixer，确保样式在不同浏览器中的一致性。
    • 代码压缩：减少 CSS 文件大小，提高加载速度。
    • 增强功能：支持现代 CSS 特性，如 CSS Grid 和 CSS Variables 的兼容性处理。

    # 典型应用场景

    • 优化 CSS 输出：压缩代码、合并文件，减少 HTTP 请求。
    • 兼容性处理：自动添加浏览器前缀，减少兼容性问题。
    • 现代特性支持：为旧浏览器提供现代 CSS 特性支持。

    # 代码示例（PostCSS）

    1. 安装依赖

      npm install postcss autoprefixer postcss-cli --save-dev

    2. 创建 PostCSS 配置文件

      module.exports = {
  plugins: {
    autoprefixer: {
      overrideBrowserslist: ['last 2 versions', '> 1%', 'Firefox ESR', 'IE 9']
    }
  }
};

    3. 编写 CSS 文件

      .button {
  display: flex;
  align-items: center;
  justify-content: center;
  padding: 10px 20px;
  border-radius: 5px;
  background-color: #3498db;
  color: white;
  cursor: pointer;
}

    4. 运行 PostCSS

      npx postcss input.css -o output.css

    # Mixins 的定义和使用

    # 概念

    Mixins 是预处理器中定义可重用样式代码块的方法。

    # 代码示例（Sass）

    // 定义 Mixin
@mixin flex-container($direction: row, $justify: flex-start, $align: center) {
  display: flex;
  flex-direction: $direction;
  justify-content: $justify;
  align-items: $align;
}

// 使用 Mixin
.container {
  @include flex-container;
}

.navbar {
  @include flex-container(row, space-between, center);
}

.card {
  @include flex-container(column, center, center);
}

    # 代码示例（Less）

    // 定义 Mixin
.flex-container(@direction: row; @justify: flex-start; @align: center) {
  display: flex;
  flex-direction: @direction;
  justify-content: @justify;
  align-items: @align;
}

// 使用 Mixin
.container {
  .flex-container();
}

.navbar {
  .flex-container(row, space-between, center);
}

.card {
  .flex-container(column, center, center);
}

    # 总结

    • 预处理器：通过变量、Mixins 和嵌套规则，提高 CSS 的可维护性和复用性。
    • 后处理器：通过自动添加浏览器前缀、优化和压缩 CSS，提高 CSS 的兼容性和性能。
    • Mixins：定义可重用的样式代码块，减少代码冗余，提高开发效率。

    这些工具和方法特别适合大型项目的开发和维护，可以显著提高开发效率和代码质量。
