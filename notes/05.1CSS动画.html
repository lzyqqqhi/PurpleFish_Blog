# CSS动画

# CSS 动画关键属性对比——animation、transition、transform

# 一、核心属性概览

  • animation：用于创建复杂的动画序列。
  • transition：用于状态发生变化时元素属性的平滑过渡。
  • transform：用于元素的二维/三维变形，如平移、旋转、缩放、倾斜等。

# 二、关键属性详解

# 1. transition

  • 作用：
    • 指定元素在状态变化时，属性如何过渡到新状态
  • 实现方式：
    • 可以设置过渡的属性、持续时间、动画方式（timing function）、延迟时间等
  • 典型场景：
    • 元素颜色、大小、位置的平滑切换

# 2. transform

  • 作用：
    • 对元素进行2D或3D变形，如移动、旋转、缩放、倾斜等
  • 实现方式：
    • 直接修改形变相关属性实现动画效果
  • 常见配合：
    • 通常与 transition 或 animation 一起使用，让形变过程更加平滑

# 三、对比总结

属性 适用场景 控制能力
animation 复杂、连续动画序列 多关键帧、序列控制
transition 状态变化的平滑过渡 单次过渡
transform 元素的形变动画 形态变换

# 四、课件式讲述

  • animation 适用于实现如加载动画、Banner 动画等复杂连续变化场景
  • transition 适用于悬停、聚焦等状态变化时的平滑过渡
  • transform 适用于对元素做平移、旋转、缩放等变形，常与前两者组合实现流畅动效

# 小结

  • animation —— 复杂动画序列
  • transition —— 状态变化时的平滑过渡
  • transform —— 元素变形处理动画

关键点记忆：

  • animation 管多阶段，transition 处理状态，transform 搞形变。
  • 场景选对，动画丝滑！

# 面试题:

# 1. CSS 动画实现的方式有哪几种？它们的区别是什么？

答案：

  1. transition（过渡）
    1. 用于属性状态的平滑过渡，需由事件（如 hover、active、JS 控制）触发
    2. 语法简洁，适合简单的状态切换动画
  2. animation（关键帧动画）
    1. 通过 @keyframes 定义的动画序列，支持更复杂的多阶段动画，可自动循环播放，无需事件触发
  3. transform（位移、旋转、缩放）
    1. 常配合 transition、animation 实现硬件加速动画
  4. 使用第三方动画库
    1. 如 Animate.css、hover.css 等，但原理也是上述两种方式

区别简表：

方式 是否自动播放 是否可循环 控制复杂程度 典型场景
transition 否 否 简单状态切换 hover 动画
animation 是 可循环 复杂/多阶段 loading 图标

# 2. 如何写一个简单的 css 闪烁动画？

答案：

@keyframes blink {
  0%,100% { opacity: 1; }
  50%     { opacity: 0; }
}
.blink {
  animation: blink 1s infinite;
}

解析：

用 @keyframes 关键帧控制 opacity 变化，animation 属性设置动画时长和无限循环。

# 3. transition 能否与 transform、opacity、color等属性一起使用？为什么建议优先用 transform/opacity 做动画？

答案：

  • 可以配合 transform、opacity、color、background、width、height 等属性使用。
  • 通常**建议优先用 **transform** 和 ****opacity** 做动画，原因如下：
    • transform、opacity 动画由 GPU 硬件加速，性能极佳，重绘/重排影响小
    • 改变位置、缩放或透明，通常不会影响页面布局（不触发 Reflow）
    • 其他属性（如 width/height/left/top等）动画，会引发重排、性能较差

# 4. 如何让一个元素等待 0.5s 再执行动画？

答案：

  • 使用 transition-delay 或 animation-delay 属性
.fade-in {
  opacity: 0;
  animation: fadeIn 1s 0.5s forwards;
}
@keyframes fadeIn {
  to { opacity: 1; }
}
  • animation: 名称 时长 延迟 其它，其中 0.5s 就是延迟时间

# 5. CSS3 animation属性有哪些常见参数？分别作用是什么？

答案：

常见属性有：

  • animation-name：关键帧名称
  • animation-duration：持续时间
  • animation-timing-function：节奏曲线（如 ease, linear, cubic-bezier 等）
  • animation-delay：延迟
  • animation-iteration-count：循环次数（无限用 infinite）
  • animation-direction：方向（normal, reverse, alternate, alternate-reverse）
  • animation-fill-mode：动画前后状态（none, forwards, backwards, both）
  • animation-play-state：是否在运行或暂停

示例：

.my-anim {
  animation: bounce 1s 0.3s linear infinite alternate both;
}

# 6. 如何实现动画暂停/继续？

答案：

  • 用 animation-play-state 属性：
.paused { animation-play-state: paused; }
.running { animation-play-state: running; }
  • 可用 JS 控制类名切换实现动画的暂停和恢复。

# 7. 榜首经典问法：transition 和 animation 具体适用场景举例

答案示例：

  • transition 适用于：按钮 hover 变色、input 聚焦、侧边栏收展等“状态切换”动画
  • animation 适用于：复杂 loading 图标、进度条、弹窗/通知自动消失动效、炫酷 Banner 动画等“连续运动”场景

# 8. 如何实现一个 loading 转圈动画？

答案：

.loading {
  width: 40px; height: 40px; border: 4px solid #ccc;
  border-top: 4px solid #2196F3; border-radius: 50%;
  animation: spin 1s linear infinite;
}
@keyframes spin {
  to { transform: rotate(360deg); }
}

面试总结答法：

CSS 动画主要有 transition 和 animation 两种方式，推荐 transform、opacity 做动画，实现高性能的交互效果。animation 适合复杂、自动、可循环的动画场景，transition 用于事件触发的简易动画。此外，理解动画参数及延迟、暂停等控制细节是面试高频考察点。
