# HTML元素专题

# 格式化

# abbr

<abbr> 标签定义缩写词或首字母缩写词，例如“HTML”、“CSS”、“Mr.”、“Dr.”、“ASAP”、“ATM”。

通过对缩写进行标记，您能够为浏览器、拼写检查和搜索引擎提供有用的信息。

<abbr> 标签最初是在 HTML 4.0 中引入的，表示它所包含的文本是一个更长的单词或短语的缩写形式。

<abbr title="World Health Organization">WHO</abbr> 成立于 1948 年。

# address

<address> 标签定义文档或文章的作者/所有者的联系信息。

如果 <address> 元素位于 <body> 元素内，则它表示文档联系信息。

如果 <address> 元素位于 <article> 元素内，则它表示文章的联系信息。

提示：<address> 元素通常连同其他信息被包含在 元素 中。

联系信息可以是电子邮件地址、URL、实际地址、电话号码、社交媒体账号等。

<address> 元素中的文本通常以斜体呈现，浏览器总是会在 <address> 元素前后添加一个换行符。

<address>
Written by <a href="mailto:webmaster@example.com">Bill Gates</a>.<br>
在这里访问我们：<br>
Example.com<br>
Box 564, Disneyland<br>
USA
</address>

# output

<output> 标签用于表示计算的结果（例如由脚本执行的计算）。

<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
  <input type="range" id="a" value="50">
  +<input type="number" id="b" value="25">
  =<output name="x" for="a b"></output>
</form>
