# HTML5新API

  1. 地理位置定位（Geolocation）
  • 定义 ：通过navigator.geolocation对象获取用户的地理位置信息，如经度、纬度等。
  • 主要方法
    • .getCurrentPosition(success, error)：获取用户当前位置，需要两个回调函数，分别处理成功和错误情况。
    • watchPosition(success, error, options)：定期获取用户位置信息，可指定选项如最大年龄、容忍度等。
    • clearWatch(watchId)：停止观察由watchPosition()启动的观察。
  • 使用场景 ：基于位置的个性化服务，如附近搜索、地图导航、地理标记等。
  • 示例代码 ：navigator.geolocation.getCurrentPosition(success, error)
  1. 拖放（Drag and Drop）
  • 定义 ：HTML5的拖放功能允许用户通过拖拽操作在页面元素之间移动内容。
  • 实现方式
    • 设置元素的draggable属性为true，使其可拖动。
    • 监听拖拽事件，如dragstart、dragover、drop等，在事件处理函数中定义拖拽行为和数据传递。
  • 使用场景 ：文件上传、元素排序、内容重组等。
  • 示例代码 ：<div draggable="true"></div>
  1. Web Worker（多线程）
  • 定义 ：Web Worker允许在后台线程中运行脚本，避免长时间运行的脚本阻塞主线程。
  • 创建和使用
    • const worker = new Worker('worker.js')：创建一个Worker线程，指定要运行的脚本文件。
    • 主线程和Worker线程通过postMessage()发送消息，通过onmessage事件监听消息。
    • 可以使用terminate()方法终止Worker线程。
  • 使用场景 ：复杂计算、网络请求、长时间运行的脚本等。
  • 注意事项 ：Worker线程不能直接操作DOM，也不能访问一些全局对象如window。
  • 示例代码 ：const worker = new Worker('worker.js')
