# Object方法

# 静态方法

  1. Object.assign(target, ...sources)
    • 用于将所有可枚举属性的值从一个或多个源对象复制到目标对象。
    • 示例：const obj = Object.assign({}, { a: 1 }, { b: 2 });
  • 快捷写法：使用展开运算符 ... 进行对象浅拷贝。
    • 示例：const obj = { ...{ a: 1 }, ...{ b: 2 } };

  1. Object.create(proto, [propertiesObject])

    • 创建一个新对象，使用现有的对象来提供新创建的对象的 __proto__。
    • 示例：const newObj = Object.create(oldObj);

  2. Object.defineProperty(obj, prop, descriptor)

    • 直接在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回此对象。
    • 示例：Object.defineProperty(obj, 'prop', { value: 42, writable: false });

  3. Object.defineProperties(obj, props)

    • 直接在一个对象上定义一个或多个新的属性，或者修改一个对象的现有属性，并返回此对象。
    • 示例：Object.defineProperties(obj, { prop1: { value: 1 }, prop2: { value: 2 } });

  4. Object.entries(obj)

    • 返回一个给定对象自身可枚举属性的键值对数组。
    • 示例：const entries = Object.entries({ a: 1, b: 2 }); // [['a', 1], ['b', 2]]

  5. Object.freeze(obj)

    • 冻结一个对象，使其属性不可修改。
    • 示例：Object.freeze(obj);

  6. Object.fromEntries(iterable)

    • 把键值对列表转换为一个对象。
    • 示例：const obj = Object.fromEntries([['a', 1], ['b', 2]]); // { a: 1, b: 2 }

  7. Object.getOwnPropertyDescriptor(obj, prop)

    • 返回指定对象上一个自有属性对应的属性描述符。
    • 示例：const descriptor = Object.getOwnPropertyDescriptor(obj, 'prop');

  8. Object.getOwnPropertyDescriptors(obj)

    • 获取一个对象的所有自身属性的描述符。
    • 示例：const descriptors = Object.getOwnPropertyDescriptors(obj);

  9. Object.getOwnPropertyNames(obj)

    • 返回一个数组，包含对象自身的所有可枚举和不可枚举属性的名称。
    • 示例：const names = Object.getOwnPropertyNames(obj);

  10. Object.getOwnPropertySymbols(obj)

    • 返回一个数组，包含对象自身的所有 Symbol 属性的名称。
    • 示例：const symbols = Object.getOwnPropertySymbols(obj);

  11. Object.getPrototypeOf(obj)

    • 返回指定对象的原型（即 __proto__ 指向的对象）。
    • 示例：const proto = Object.getPrototypeOf(obj);

  12. Object.is(value1, value2)

    • 比较两个值是否相同。
    • 示例：Object.is(25, 25); // true
  • 对于严格相等比较，可以使用 === 运算符。
    • 示例：25 === 25; // true

  1. Object.isExtensible(obj)

    • 判断一个对象是否可扩展（即是否可以添加新的属性）。
    • 示例：Object.isExtensible(obj);

  2. Object.isFrozen(obj)

    • 判断一个对象是否被冻结。
    • 示例：Object.isFrozen(obj);

  3. Object.isSealed(obj)

    • 判断一个对象是否被密封。
    • 示例：Object.isSealed(obj);

  4. Object.keys(obj)

    • 返回一个包含对象自身所有可枚举属性的名称的数组。
    • 示例：const keys = Object.keys(obj);

  5. Object.preventExtensions(obj)

    • 防止一个对象被扩展，即不能再添加新的属性。
    • 示例：Object.preventExtensions(obj);

  6. Object.seal(obj)

    • 密封一个对象，使其属性不可添加和删除。
    • 示例：Object.seal(obj);

  7. Object.setPrototypeOf(obj, prototype)

    • 设置一个对象的原型。
    • 示例：Object.setPrototypeOf(obj, newProto);

  8. Object.values(obj)

    • 返回一个包含对象自身所有可枚举属性值的数组。
    • 示例：const values = Object.values(obj);

# 实例方法

  1. obj.hasOwnProperty(prop)
    • 判断一个对象是否包含特定的自身（非继承）属性。

  • 示例：obj.hasOwnProperty('prop');

    • 可以使用

       Object.prototype.hasOwnProperty.call(obj, prop)

      作为替代。

    • 示例：

      Object.prototype.hasOwnProperty.call(obj, 'prop');

  1. obj.isPrototypeOf(obj2)

    • 判断一个对象是否是另一个对象的原型。
    • 示例：obj.isPrototypeOf(obj2);

  2. obj.propertyIsEnumerable(prop)

    • 判断一个对象上的属性是否可枚举。
    • 示例：obj.propertyIsEnumerable('prop');

  3. obj.toString()

    • 返回对象的字符串表示。
    • 示例：obj.toString();

  4. obj.valueOf()

    • 返回对象的原始值。
    • 示例：obj.valueOf();
