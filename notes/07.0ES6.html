          # ES6

          # 变量

          # var 的问题

          • 变量提升：声明会提升到作用域顶部
          • 没有块级作用域：容易污染全局或外层作用域
          • 可重复声明与任意修改：安全性差

          # let

          • 块级作用域，声明变量可重新赋值，但不能重复声明
          • 仅在当前代码块内有效
          if (true) {
  let i = 1;
  console.log(i); // 1
}
console.log(i); // ReferenceError

          # const

          • 块级作用域，只读常量，声明必须初始化
          • 常量指向绑定不变（对于对象属性，值可变）
          const PI = 3.14;
// PI = 3; // Error

const user = { name: "张三", age: 18 }
user.name = "李四"; // 合法，指向的对象内容可变
Object.freeze(user); // 完全冻结对象

          # 实践建议与原理

          • 推荐优先 const（默认不可变，更安全），仅变量需改变时用 let
          • 编译原理：let/const 创建于块级作用域产生的词法环境（Lexical Environment），存在“暂时性死区”（TDZ），只有执行到声明时才能被访问和赋值。

          # 面向对象编程 —— class 语法

          # 类与构造函数

          • class 是语法糖，底层还是构造函数+原型
          class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}
const p = new Person("Tom", 20);
          • 类的方法默认不可枚举（for...in 不遍历）

          # 继承

          class Student extends Person {
  constructor(name, age, grade) {
    super(name, age);
    this.grade = grade;
  }
}

          # 静态属性和方法

          class Person {
  static species = "human";
  static saySpecies() {
    console.log(`We are ${this.species}.`);
  }
}
Person.saySpecies(); // 静态

          # getter、setter

          class Person {
  get name() { return this._name; }
  set name(val) { this._name = val; }
}

          # class 表达式

          const Person = class { ... }

          # 模板字符串

          • 用反引号 ```
          • 支持多行、变量嵌入 ${}、表达式，提升可读性
          const name = "Tom", age = 20;
const str = `My name is ${name}, I'm ${age} years old.`;

          # 标签模板（tagged template）

          • 可以自定义处理模板
          function up(strings, ...values) {
  return strings.reduce((s, cur, idx) => s + (values[idx-1] && String(values[idx-1]).toUpperCase() || '') + cur);
}
up`hello ${'tom'}, age ${18}`; // 'hello TOM, age 18'

          # 箭头函数

          # 特点

          • 语法更简洁：(...args1) => a + b
          • 没有自己的 this，arguments，不可 new，无法 call/apply/bind 改变 this
          • this 固定为外部作用域
          • 常见的判断箭头函数打印题目笔记 不同的调用方式 this指向 console打印

          # 场景

          1. 事件、定时器、异步回调（需绑定 this 时）
          2. 简写函数式编程 map/filter/reduce
          3. mao实现reduce ，foreach实现map

          # 限制

          • 不能做构造函数
          • 不能使用 arguments
          • this 不能被显式修改

          # 生成器（Generator）

          • 解决函数不能中断执行的问题
          • function* 生成器，yield 暂停，next() 恢复
          function* gen() {
  yield 1;
  yield 2;
}
const g = gen();
g.next().value; // 1
g.next().value; // 2

          # BigInt

          • 转字符串 1.233 + 0.3223 大数相加 大数相乘
