      # 本地存储

      1. localStorage

        1. 特点：持久存储，除非手动清除，否则数据永久存在。

        2. 用法示例：

          • // 保存数据
localStorage.setItem('key', 'value');
// 获取数据
const value = localStorage.getItem('key');
// 删除数据
localStorage.removeItem('key');
// 清空所有数据
localStorage.clear();

      2. sessionStorage

        1. 特点：仅在当前会话（浏览器窗口/标签页）有效，关闭窗口即被清除。
        2. 用法与 localStorage 相同。

      # Cookie

      Cookie 是一种在你浏览网页时，网站存储在你设备上的小文本文件。它主要用于会话跟踪、个性化体验、记住用户信息和网站分析等。

      # Cookie 的基本结构和使用方式

      • 结构 ：cookie 是一个简单的文本文件，其中包含了键值对形式的数据。例如：username="john_doe"; password="encrypted_password"; preferences="dark_mode"。
      • 设置 cookie ：网站通过 HTTP 响应头中的 Set-Cookie 字段来设置 cookie。
      • 读取 cookie ：网站通过 HTTP 请求头中的 Cookie 字段来读取 cookie。
      • 删除 cookie ：网站可以通过设置 cookie 的过期时间来删除 cookie。

      # 代码

      # 解码 Cookie 的 Python 示例代码

      以下是一个使用 Python 对 cookie 进行解码的示例代码。这个示例演示了如何从服务器的响应头中提取 cookie，并将其解析为一个字典格式。

      from http.cookies import SimpleCookie

# 示例响应头中的 Set-Cookie 字符串
set_cookie_header = "username=JohnDoe; path=/; HttpOnly; Secure, preferences=dark_mode; path=/; SameSite=Lax"

# 创建 SimpleCookie 对象
cookies = SimpleCookie()

# 使用 load 方法加载 Set-Cookie 字符串
cookies.load(set_cookie_header)

# 将 cookie 解码为字典格式
decoded_cookies = {}
for key, morsel in cookies.items():
    decoded_cookies[key] = morsel.value

print(decoded_cookies)

      # 解码 Cookie 的 JavaScript 示例代码

      以下是一个使用 JavaScript 对 cookie 进行解码的示例代码。这个示例演示了如何从客户端的 document.cookie 中提取 cookie，并将其解析为一个对象格式。

      // 获取 document.cookie 中的 cookie 字符串
const cookieString = document.cookie;

// 将 cookie 字符串解码为对象格式
const decodedCookies = cookieString.split(';').reduce((acc, cookie) => {
  const [key, value] = cookie.trim().split('=');
  acc[key] = decodeURIComponent(value);
  return acc;
}, {});

      # 设置 Cookie 的代码示例

      1. Python（服务器端 - Flask 框架）
      from flask import Flask, make_response

app = Flask(__name__)

@app.route('/set-cookie')
def set_cookie():
    response = make_response("Cookie 设置成功！")
    # 设置一个 cookie，名称为 'username'，值为 'JohnDoe'
    response.set_cookie('username', 'JohnDoe', path='/', httponly=True, secure=True)
    return response

if __name__ == '__main__':
    app.run()

      # 对比

      对比点 localStorage / sessionStorage cookie
      储存大小 约 5MB（各浏览器略有差异） 约 4KB
      生命周期 localStorage 永久
      sessionStorage 会话      		 可设置过期时间
      与 HTTP 请求 不会自动随请求发送 每次请求会自动携带
      操作接口 JS API，简洁易用 需要手动字符串拼接或用库
      用途 前端数据缓存，增强型离线应用等 通常用于会话管理、登录信息等
      安全性 相对更安全（同源策略限制访问） 易被抓包、劫持，有安全风险
      服务端能否操作 否（只能前端读写） 是（服务器通过 Set-Cookie 设置）
